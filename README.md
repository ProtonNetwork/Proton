## Development Stage One: Ari ##

### Setup Firewall ###
* Open Port:37000


### Install Nodejs ###
* require(version[node]>=9.11.1)


### Install Parity ###
* require(version[parity]>=1.11.0)


### Install Geth ###
* require(version[geth]>=1.8.2)


### Join Proton Chain ###
```
npm i -g pttnormal
```

### Account for PTTChain ###
* Create a new account and put it into PTTChain folder


### Fire up ###
```
protongo
```
